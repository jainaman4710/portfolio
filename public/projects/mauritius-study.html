01 / Project Overview

What this study set out to answer

The study investigates how digital literacy intersects with student motivation (Attention, Confidence, Relevance, Satisfaction) and blended learning course experience, using five interconnected research questions, each answered with a distinct statistical method.

Context
Higher education students in Mauritius enrolled in blended learning courses. Survey data collected across two universities.
Theoretical framework
ARCS motivation model (Keller): Attention, Relevance, Confidence, Satisfaction, cross-validated with digital literacy and connected/separated learning attitudes.
Survey instrument
50 Likert-scale items across 10 composite constructs: DL, CD, LE, PF, A, R, S, C, CL, SL. Composite scores computed before modelling.
Role in this project
End-to-end data analyst: data cleaning, dimensionality validation, EFA/CFA, regression, SEM specification, fit evaluation, and interpretation.

🔬
Exploratory & Confirmatory Factor Analysis
Used oblimin rotation to identify latent factor structure across 10 composite indicators. Validated with CFA using lavaan's 4-factor theoretical model.
CFI = 0.962–0.973 across models
📊
One-Way ANOVA
Compared motivation construct means between high and low digital literacy groups (median split). Reported generalised eta-squared (ges) as effect size measure.
F-stats range: 43.51–97.84
🧮
Stepwise Regression
Applied forward stepwise selection to identify the minimal set of predictors with significant unique variance in connected learning attitude.
Adj. R² = 0.254 · p < .001
🔗
Mediation Analysis (SEM)
Specified and fitted multiple SEM models to test both direct and indirect pathways. Digital Literacy confirmed as significant mediator between motivation and course experience.
Indirect effect p < .001
📐
Model Fit Evaluation
All SEM models evaluated on CFI (>0.95 = good) and SRMR (<0.05 = good). Multiple competing model specifications tested before final selection.
Best fit: CFI=0.973 / SRMR=0.028
🛠
R + lavaan Implementation
Full analysis pipeline in R Markdown: data loading (readxl), factor analysis (psych), CFA/SEM (lavaan), visualisation (ggplot2), and reproducible reporting.
~900 lines of annotated R code

This is the study's flagship finding. The standardised path coefficient of β = 0.928 represents a near-perfect linear relationship; course experience explains 86.1% of the variance in student motivation. The implication: course design quality is the primary driver of student motivation in blended learning environments.
